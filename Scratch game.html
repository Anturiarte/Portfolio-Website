Scratch Game

Brief Description

For my Scratch Project, I had been working with Sebastion Lopez and Izaya Zuniga. I mainly did the code for the game, Sebastion did the graphics for the sprites, and Izaya helped with both. Our game is an 8-bit style game inwhich you are a kid with a sword who has to kill enemies and each enemy has a special ability. The kid is able to interact with some of the enemies as well which is pretty cool.
